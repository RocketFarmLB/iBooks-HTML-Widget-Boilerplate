Scrolling iBooks HTML Widget Boilerplate

This is a simple boilerplate for an iBooks widget which scrolls vertically (or horizontally should you wish), functionality which *should* already exist.

Credits

  • Standing on the shoulders of others - Ed Wilde
  • Using the superb JQuery Custom Content Scroller by malihu
  • Included JQueryUI library is customised to remove unused features, feel free to replace it should you need the full set.
  • Boilerplate was developed from original work by Trevor Burnham, open-sourced under the MIT License.
